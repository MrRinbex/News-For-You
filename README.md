# [World Wide News Today](https://world-wide-news-today.netlify.app)

<img src="./public/logo.png" />

Package & Framework using :

- Next.js 13
- TypeScript
- Tailwindcss
- StepZen
- GraphQL
- Mediastack
- Netlify
- Timeago
- React
- Dark Mode

The API is limited by 500 requests monthly
